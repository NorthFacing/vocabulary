# freakishly 

* ['fri:kiʃli]

* adv. 反常地，奇特地

### 相关词

* freakish ['friːkɪʃ] 
    - adj. 畸形的；奇特的；朝三暮四的；异想天开的
* freaky ['fri:ki] 
    - adj. 畸形的；捉摸不定的（等于freakish）
* freak [friːk] 
    - n. 怪人，怪事；畸形人；反复无常
    - adj. 奇异的，反常的

### 例句

```
The truth is that the architecture of most language applications is __freakishly__ similar.
—— Language Implementation Patterns, P12
```
